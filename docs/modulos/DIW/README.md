# 🎨 DIW - Disseny d'Interfícies Web

## 📋 Descripción del Módulo

El módulo **DIW (Disseny d'Interfícies Web)** se especializa en el diseño y desarrollo de interfaces de usuario profesionales, accesibles y responsivas para la aplicación e-commerce AlberoPerezTech. Implementa sistemas de diseño completos, componentes UI reutilizables, y garantiza una experiencia de usuario óptima siguiendo los principios de UX/UI modernos y los estándares WCAG de accesibilidad.
