# 🤖 DIG - Digitalització

## 📋 Descripción del Módulo

El módulo **DIG (Digitalització)** se especializa en la implementación de tecnologías inteligentes y digitales que mejoran la experiencia del usuario y optimizan los procesos de negocio en la plataforma e-commerce AlberoPerezTech. Implementa sistemas de recomendación, automatización de procesos con n8n, integración de servicios externos y digitalización de la atención al cliente.
